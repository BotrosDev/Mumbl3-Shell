Mumbl3-Shell has a unique top hidden bar called " dropdownWindowPanel " in the Quickshell files, the dropdownPanel directory has everything inside. 
The " Colors.qml " found inside " Data " folder is the main colors file. and there is no other color files. the Colors.qml collect the colors made by the matugen 
the matugen then generates a Json file in the " ~/.local/share/quickshell" folder. then the colors.qml file collect the data and drops it as qml language.

There are several files unused in the Data folder. as the entire folder was made by the same auther of frienorainobar. i Did ask for permission PERSONALY to use their code.
The core idea of the dropdown Panel was not mine. But the work Done was mine and i did modify some of the original auther of frinorainobar and some stuff are entirly rebuilt by me all from scratch. 

Images and Gifs are in ~/.config/quickshell/bar/Assets

    click on clock to reveal full date.
