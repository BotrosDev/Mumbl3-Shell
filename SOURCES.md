# Asset Sources Guide
This shell uses a Frieren: Beyond Journey's End theme. Due to copyright, character artwork is not included.
You'll need to add your own images to complete the theme!

# Required Assets
Place these files in `~/.config/quickshell/bar/Assets/:`

[Kuru Kuru frieren](https://tenor.com/view/frieren-kuru-kuru-gif-6387185842025446472) `required in media player`
[friedance](https://tenor.com/view/frieren-pokedance-frieren-dancing-frieren-dance-gif-18034940161722560799) `required by media player`

[frei&himmel](https://cdn.donmai.us/original/c2/71/c27125310e6fbafaf16294c54019be20.png) `not required`

[chibbiSource](https://www.reddit.com/media?url=https%3A%2F%2Fpreview.redd.it%2Fchibi-party-v0-me8ysq7p4iob1.jpg%3Fauto%3Dwebp%26s%3De7e86ae3e1a57aca1c38cb6b2ec52b48ba51bcf1)
`modified, the chibis are used in clock and you dont need to cut it from the source`

[seriesit](https://s1.zerochan.net/Serie.(Sousou.no.Frieren).600.4126160.jpg) `required in calendar`

[Frieren](https://pnganime.com/gallery/frieren-menacing-transparent-png) `required for dropdown Panel default image (optional)`

# Alternative Themes
Don't Want Frieren assets? Use a different anime or create your own theme!
The shell works with ANY images.
